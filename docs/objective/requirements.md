# Szorgalmi programozási feladat

- Határidő: 2025. november 30. 23:59
- Lezárás: 2025. december 7. 23:59

## Utasítások
Kérem töltse fel a szorgalmi feladathoz elkészített program forráskódját és a dokumentációt!

## A dokumentációnak az alábbi részeket kell tartalmaznia
- címlap
- tartalomjegyzék
- a feladat leírása (kb. 0,5–1 oldal)
- a megoldás elve, módszere (kb. 0,5–1 oldal)
- az implementáció bemutatása az adott programozási környezetben (kb. 1–2 oldal)
- a program futtatásával nyert tapasztalatok ismertetése, a bemeneti változók hatásainak vizsgálata esettanulmányokon keresztül (pl. futási idő, memóriafelhasználás, az eredmények reprodukálhatósága, stb.; kb. 2–4 oldal)

> A program személyes bemutatásával kapcsolatosan kérem egyeztessenek a feladatért felelős tanszéki kollégával.
